# 4D-polytope

Auxiliary files for the paper "Conditions for boundedness from below of a Δ(54)-symmetric three-Higgs-doublet model" in which we determin the vertices, edges, and faces of the four-dimensional polytope.

- The Mathematica notebook with code ["Polytope.nb"](https://github.com/jurciukonis/4D-polytope/blob/main/polytope.nb).

- The edges, faces, and facets of the four-dimensional polytopede are given in ["polytope.pdf"](https://github.com/jurciukonis/4D-polytope/blob/main/polytope.pdf).
